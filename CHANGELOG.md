## [0.1.0] - 2024-01-XX

### Added
- Initial release
- Web-based investigation dashboard
- Cross-platform agent support (Windows, Linux, macOS)
- Windows binary (.exe) support
- Windows kernel-level agent
- Domain name detection and tagging
- Computer name and IP address collection
- Classic forensic interface hex viewer
- File system browser with breadcrumb navigation
- Metadata display panel
- Real-time WebSocket communication
- Chunked file reading for large files
- MD5 hash calculation
- Search functionality in hex viewer

### Features
- Agent management dashboard
- Remote file system browsing
- Forensic file viewer (hex view + text view)
- Comprehensive file metadata
- Multi-IP address support
- Domain-aware agent identification

### Security
- ⚠️ No authentication (development only)
- ⚠️ No encryption (HTTP only)
- ⚠️ No access controls

### Known Limitations
- Requires manual agent deployment
- No automated updates
- No audit logging
- No case management
